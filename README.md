# Ping



* Compile with:
    * gcc -Wall ping.c -o ping -pthread
    * or: make ping

* Usage:
    * ./ping <hostname or IP>
    * ./ping localhost
    * ./ping www.google.com
    * ./ping 8.8.8.8

* If having permission issue:
    * try running as the root user
* On MacOS, try:
    * sudo ./ping <hostname or IP>
